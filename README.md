# Helium

Helium is an experimental OS written in C++ targeting my personal Intel Pentium 4 PC. Helium is experimental, currently not ready for serious use.

### The Kernel

The kernel is monolithic, developed mostly in C++ and only targeting x86. The kernel is also *POSIX-like* in nature, with a Linux-like ABI. This allows for fairly easy porting of applications to the project target as the project is developed.

> [!NOTE]
> This is *highly* unstable, do not expect this to work immediately in testing!
